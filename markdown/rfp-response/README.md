# RFP Response
## Overview 
These are the prompts I use to assess and respond to RFPs.
- **0-instructions.md**: Mast prompt to use with the other three.
- **1-assess-rfp.md**: Compact prompt to assess the RFP and our related experience.
- **2-proposal-draft.md**: Compact prompt to create a rough draft of the technical volume.
- **3-red-team.md**: Compact prompt to have the LLM red team the technical volume.

## Use
### ChatGPT
I originally created these for use with ChatGPT. To use them:
1. I create a new project.
2. I copy the contents of 0-instructions.md into the "Instructions" field from the "Project Settings" panel.
3. I upload the other three files to the project's sources.
